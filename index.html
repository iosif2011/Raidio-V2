Setup required

This player reads songs from a songs/ folder via a manifest file. Set up your repo like this:

1

Create the folder structure

your-repo/ ├── index.html ├── songs.json └── songs/ ├── my-track.mp3 ├── another.wav └── ...
2

Create songs.json listing every file

[ "my-track.mp3", "another.wav", "cool-song.flac" ]
3

Push to GitHub and enable Pages

Go to Settings → Pages, set source to the branch containing these files. The site will auto-sync across all devices that open it.
